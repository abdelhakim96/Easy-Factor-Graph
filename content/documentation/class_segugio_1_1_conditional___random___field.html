Detailed Description

This class describes Conditional Random fields.

Set_Observation_Set_var is depracated: the observed set of variables cannot be changed after construction.

Returns an estimation of the likelihood of the model.

(weights describing the wrapped Potential_Exp_Shape), considering a particular training set as reference: P(model | train_set). This method is called by an I_Trainer during the gradient descend performed when training the model

Parameters
[in]comb_train_setsamples contained in a training set, obtained calling Training_set::subset::Handler::Get_list
[in]comb_var_orderlist of variables describing how the values in comb_train_set are ordered (they must refere to the variables wrapped by this model, considering both the hidden and the observed sets).
[out]resultlogarithmic estimation of the likelihood
