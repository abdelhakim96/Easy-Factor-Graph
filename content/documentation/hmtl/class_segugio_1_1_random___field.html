Parameters
[in]potentials_expthe initial set of exponential potentials to insert (can be empty)
[in]use_cloning_Insertwhen is true, every time an Insert of a novel potential is called (this includes the passed potentials), a copy of that potential is actually inserted. Otherwise, the passed potential is inserted as is: this can be dangerous, cause that potential cna be externally modified, but the construction of a novel graph is faster.
[in]tunable_maskwhen passed as non default value, it is must have the same size of potentials. Every value in this list is true if the corresponfing potential in the potentials list is tunable, i.e. has a weight whose value can vary with learning
[in]shapesA list of additional non learnable potentials to insert in the model
